üåê API Endpoints

Role Management Endpoints:

  • POST /api/users-roles-management/roles - Create role
  • GET /api/users-roles-management/roles - Get all roles
  • GET /api/users-roles-management/roles/active - Get active roles
  • GET /api/users-roles-management/roles/{id} - Get role by ID
  • PUT /api/users-roles-management/roles/{id} - Update role
  • DELETE /api/users-roles-management/roles/{id} - Delete role
  • POST /api/users-roles-management/roles/{id}/activate - Activate role
  • POST /api/users-roles-management/roles/{id}/deactivate - Deactivate role

User Role Assignment Endpoints:

  • POST /api/users-roles-management/assign-role - Assign role to user
  • PUT /api/users-roles-management/user/{userId}/role/{newRoleId} - Update user role
  • DELETE /api/users-roles-management/user/{userId}/role/{roleId} - Remove role from user
  • GET /api/users-roles-management/user/{userId}/roles - Get user roles
  • GET /api/users-roles-management/user/{userId}/permissions - Get user permissions

Permission Check Endpoints:

  • GET /api/users-roles-management/user/{userId}/has-permission/{moduleName}/{permission} - Check specific permission
  • GET /api/users-roles-management/user/{userId}/accessible-modules - Get accessible modules
  • GET /api/users-roles-management/user/{userId}/has-access/{moduleName} - Check module access

üéØ Default Roles & Permissions

SUPER_ADMIN:

  • ‚úÖ Full access to all modules
  • ‚úÖ All permissions (ADD, VIEW, EDIT, DELETE)
  • ‚úÖ Can manage other admins and system configuration

ADMIN:

  • ‚úÖ Limited access to most modules
  • ‚úÖ No delete permissions for critical modules
  • ‚úÖ Can manage users and basic configuration

MANAGER:

  • ‚úÖ Read/Edit access to assigned modules
  • ‚úÖ No delete permissions
  • ‚úÖ Can view analytics and manage farmers

EMPLOYEE:

  • ‚úÖ Limited access to farmer and FPO modules
  • ‚úÖ Basic permissions (VIEW, some EDIT)
  • ‚úÖ Cannot access configuration or user management

FARMER:

  • ‚úÖ Self-service access only
  • ‚úÖ VIEW and EDIT own data
  • ‚úÖ Cannot access other users' data
